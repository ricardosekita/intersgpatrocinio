# Inter SG — Patrocínio

Site estático com o material de patrocínio do Inter SG.

## Páginas
- `index.html` — landing com o menu
- `deck.html` — Deck de Patrocínio (apresentação)
- `tour.html` — Tour da Marca (vídeo interativo)

## Publicar no GitHub Pages
1. Suba o conteúdo desta pasta para a raiz do repositório (ou para uma branch `gh-pages` / pasta `/docs`).
2. No repositório, vá em **Settings → Pages**.
3. Em **Source**, selecione a branch e a pasta onde estes arquivos estão (`/root` ou `/docs`).
4. Salve — o site ficará disponível em `https://<usuario>.github.io/<repositorio>/`.

Nenhuma etapa de build é necessária — é HTML/CSS/JS puro.
